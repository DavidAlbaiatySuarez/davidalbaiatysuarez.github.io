  • Amazon similarity algorithm in python

    This algorithm is similar to the real-world similarity algorithm used by Amazon but more simplistic and with fewer number of users: my primary aim with this project was not to produce an efficient program but rather a functional piece of code to practice all the python concepts learnt throughout the year.

    To put it simple, we have a finite number of books and each user has a score for each of books (0 if the book was not read) as a vector. If the user IS NOT in ratings.txt, we will ask the new user to rate 5 different books giving us a new similarity algorithm.

    Using the dot product, we will then calculate the similarity of all users to one specific user and give them x number of recommendations. The higher (non-negative) score, the similar the users are. We output the result to a .txt file.

  • Concurrent Dependency Discoverer in C++

    The aim of this project was to implement a safe, multithreaded dependency discoverer program in C++. We want a program that can crawl over source files, keeping track of any #include directives, go through files specified in #include directives, and finally generate the correct dependency specifications.

    Since we are using threads, we need to make sure that multiple threads cannot access common, critical variables at the same time, to prevent deadlocks and the program from crashing/malfunctioning. This program is meant to be run in a Linux distribution (i.e.: Ubuntu). Note, a 'make' file is provided.

    Download 'Concurrent Dependency Discoverer'
